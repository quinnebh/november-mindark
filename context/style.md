## Styling Framework
Use Tailwinds for styling

## Look and Feel
- modern luxurious
- glassmorphism
- minimalistic
- dark theme
- dark mode by default
- gradient backgrounds
- sharp, minimal rounding

## Custom Color Palette
- Brand color: #ECBE07
- Dark scales: (50-900) for consistent theming
- Glass colors for translucent overlays
- Border colors with varying opacity levels
- Semantic text colors (primary, secondary, muted, etc.)

## Typography System
- Hero text styles with gradient effects
- Responsive sizing (72px desktop, 48px mobile)
- Letter spacing and weight variations
- Subtitle and caption utility classes

## Advanced Features
- Backdrop blur utilities (xs through xl)
- Float animations for geometric accents
- Shimmer effects for interactive elements

## Rules
Use the brand color sparingly. The brand color should only be used for the most important elements.
The brand color can be used as a thin border color in situations where an element should be highlighted
but is not a main CTA.
